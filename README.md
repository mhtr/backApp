## Currently Supported Platforms
  - Windows
  - Linux

## Requirements
  - CMake 3.9 +
  - Qt 5.10.1 +

## Source download and compilation
```sh
$ git clone https://github.com/mhtr/backApp && cd backApp
$ mkdir build && cd build
$ cmake3 -DCMAKE_PREFIX_PATH:STRING=$QTDIR .. && cmake3 --build .
$ ./backApp
```
